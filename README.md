# **This is the front-end of the Something-Todo application**

* **App** - https://something-todo.vercel.app/
* **Back-end** - https://github.com/okatsuo/something-todo

<br>

> ## Functionalities:

* Register
* Login
* Logout
* New task
* Complete task
* Delete task
